-g --> a nivel global en toda la maquina

cd ~ --> directorio de usuario.

explorer . --> te abre el directorio de la home.

ssh-keygen --> seguriza la comunicacion entre tu pc y git.

ng --version --> version de Angular
ng --help --> lista de comandos disponibles.

ng serve --> empaqueta todo el codigo y lo sirve en un server dev.
ng generate component "nombre" --> genera, un componente llamado. Se puede poner: g c
